# Linus Lucas Rwechoka — AI Engineer Portfolio

A premium, interactive portfolio showcasing projects in AI, Machine Learning, and Embedded Systems.

## 🚀 Key Features

- **Matrix Rain Background**: Dynamic, high-performance canvas animation.
- **Independent AI Representative**: A standalone, bilingual (English/Kiswahili) AI built directly into the site to answer questions about my work.
- **Project Showcase**: Detailed overview of my YOLOv5, IoT, and Database projects.
- **Glassmorphic Design**: Modern, sleek UI with smooth transitions and micro-animations.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Animations**: Canvas API, Intersection Observer API
- **AI Logic**: Standalone keyword-based response engine

## 🤖 AI Representative (Bilingual)

The portfolio features an AI rep that knows Linus's work inside out. It operates in two modes:
- **Standalone Mode**: Fast, keyword-based responses (offline).
- **Gemini Online Mode**: Powered by Google Gemini for deep, human-like conversations.

### How to Activate Online Mode
1. Click **"Go Online"** in the AI Brain header.
2. Paste your **Gemini API Key**.
3. Click **ACTIVATE**.
*Note: Your key is only active for the current session and is NOT stored anywhere.*
