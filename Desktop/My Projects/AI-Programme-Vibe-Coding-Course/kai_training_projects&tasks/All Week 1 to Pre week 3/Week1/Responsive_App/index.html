import math def calculate_risk(obj1, obj2): """Level 7: Proximity-based risk""" c1 = [obj1['bbox'][0] + obj1['bbox'][2]/2, obj1['bbox'][1] + obj1['bbox'][3]/2] c2 = [obj2['bbox'][0] + obj2['bbox'][2]/2, obj2['bbox'][1] + obj2['bbox'][3]/2] dist = math.sqrt((c1[0]-c2[0])**2 + (c1[1]-c2[1])**2) if dist < 100: return "HIGH", dist elif dist < 300: return "MEDIUM", dist else: return "LOW", dist

Feature 🇰🇷 Korean AI Standard 🇹🇿 Tanzanian Reality Impact

🚌 Daladala Focus

THE AI DOESN'T
KNOW DALADALA

The Korean model sees a bus. But Daladala is not a bus. It's culture, chaos, and community on wheels — and teaching AI to understand that is our mission.

🎨
Custom Paintwork

Vivid murals confuse object detection. AI misreads decals as separate objects.

👥
Hanging Passengers

"Person" bboxes overlapping vehicle >50% need is_overloaded: true flag.

📍
Informal Stops

No fixed stops. AI prediction models must learn to expect sudden braking anywhere.

🔤
Re-Labeling Script

Phase 1: Rename all "Bus" labels to "Daladala" in JSON files for local model training.

ROADMAP

RE-LABELING THE AUDIT

Take Korean "Bus" annotation data and run a script to universally rename labels to "Daladala" in all JSON files. Collect 50+ local Daladala images to retrain the visual detection layer with real Tanzanian context.

LOGIC ADAPTATION

Update Level 4 (Attribute Extraction) to handle overloaded Daladalas. Add boolean attribute is_overloaded when "Person" bounding boxes overlap >50% with a vehicle box. This teaches the AI to recognize cultural loading norms as safety data, not noise.
