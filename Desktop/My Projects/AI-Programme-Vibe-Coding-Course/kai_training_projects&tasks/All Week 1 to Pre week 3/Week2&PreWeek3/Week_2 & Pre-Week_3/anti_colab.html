Antigravity IDE & Colab GPU Setup Guide

This guide covers how to install the Antigravity IDE, connect to Google Colab GPU resources, and configure your Gemini API to train our Tanzanian local transport (Bajaj, Bodaboda, Daladala) classification model.

Step 1: Download & Install Antigravity IDE

  • Download the installation file suitable for your operating system (Windows, Mac, etc.) from the official Antigravity distribution site.
  • Run the downloaded file to complete the installation, and then launch the application.

Step 2: Install the Google Colab Extension

  • Click the Extensions icon on the left sidebar of Antigravity.
  • Type Google Colab in the search bar and install the official extension.

Step 3: Connect the GPU Runtime (CRITICAL)

This is where we request a dedicated AI Training GPU instead of a standard CPU.

  1. Click the Colab button located in the center of the top toolbar.
  2. Select the Connect to Google Colab... menu.
  3. When the Select a variant window appears, you MUST select GPU.
  4. Complete the Google account login and authorization popup.

Step 4: Verify Your Connection

Check if the kernel name in the top right corner has changed to Python 3 (Google Colab). Then, run the following code in the first cell to verify your hardware:

!nvidia-smi

If you see a table displaying your assigned GPU information (like Tesla T4), you have succeeded!

Step 5: Configure the Gemini API (Provided by Instructor)

To prevent hitting the usage limits of the default model, we will register the API key provided by the instructor to access the highest performance model.

  1. Click the Model Name (e.g., Gemini 3 Flash) located just above the agent chat input box.
  2. Go to the settings at the bottom of the list, or navigate to the 'Account' tab in the sidebar.
  3. Find the Google section and paste the API Key you received from the instructor.
  4. Go back to the model selection window and select Gemini 1.5 Pro (or Gemini 3 if instructed).

Step 6: Your First Command (Mount Google Drive)

Now, we command our AI Agent to prepare the workspace for our training data.

Copy and paste this exact prompt into the Agent chat box:

"The GPU connection and Gemini API setup are complete. Could you write and execute the Python code to mount my Google Drive to this server so we can load our training data?"

⚠️ PM Security Rules & Precautions

  • File Extension: You MUST be working inside a .ipynb (Jupyter Notebook) file for the Colab menus to activate.
  • Session Timeout: If you are inactive for a certain period, Google will disconnect your GPU. Keep an eye on it!
  • Security: NEVER save the provided API key as plain text inside your notebook files. Keep it strictly in the Account settings.
