# 🇹🇿 Bongo Traffic AI - Hackathon Edition 2.0

Winning submission for the **UAUT Ultimate AI Hackathon**. A world-class vehicle classification system built with OOAD principles and Gemini Vision AI.

## 🏆 Project Goal
To accurately track and classify Dar es Salaam's transport icons:
- **🛺 Bajaj**
- **🚌 Daladala**
- **🏍️ Bodaboda**

## 🏗️ Technical Excellence
- **Architecture**: Service-Repository pattern (Clean Code).
- **AI**: Google Gemini 1.5 Flash Vision.
- **UI**: High-end Glassmorphism Design System.
- **Backend**: FastAPI with full Dependency Injection.

## 🚀 Hackathon Ready
This system is optimized for zero-shot classification on unseen Dar es Salaam traffic footage. It uses a specialized reasoning engine to explain its classification choices based on color, shape, and context.

## 🛠️ Tech Stack
- FastAPI, Supabase, React (ESM), Tailwind CSS, Gemini AI.
